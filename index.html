HOW TO PLAY

Goal

Repeat the sequence of colors and sounds that Simon plays. Each round adds a new note to the sequence. See how far you can go!

Gameplay

  1. Press Start to begin.
  2. Watch and listen as the pads light up.
  3. Repeat the sequence by pressing the pads in the same order.
  4. Each correct round adds another note.
  5. One wrong press and the game is over!
